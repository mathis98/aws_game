## S3
Amazon S3 (Simple Storage System), ist ein Datenbanksystem welches von Amazon selbst
entwickelt worden ist und auch zur verfügung gestellt wird.
Dateien werden als Objekte dargestellt und mit einem eindeutigem Key vom User
gekennzeichnet. Objekte werden in Buckets (auch mit eindeutigen Namen) gruppiert.
Objekte können bis zu 5 Terabyte groß sein.
Buckets können einzeln adressiert werden und die darin enthaltenen Daten mittels HTTP
GET oder BitTorrent-Protokoll heruntergeladen werden.
